Fill & Sign

Type into the fields on the document. Click a Mock Sign or E Sign box to sign it -- that locks the document at that point, but other unsigned fields can still be filled and signed afterward.

Add Field

Click a type, then click where you want it. Double-click a type to keep placing several in a row. Press Esc to cancel.

Arrange

Shift-click fields to select more than one.

Inspector

Conditional Logic

Build if / then / else logic visually. Rules are checked in order; the first match wins. Use another computed field as a source field for nested logic.

Custom Script (Advanced)

Real Acrobat JavaScript embedded into the exported PDF. Runs in PDF viewers that execute Acrobat JavaScript, such as Acrobat Reader or Acrobat Pro. Custom scripts override Format, Calculation, and Conditional Logic above when both are set. See Adobe's Acrobat JS API reference.
